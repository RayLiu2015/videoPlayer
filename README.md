# videoPlayer:
一个iOS平台上基于AVPlayer封装的播放器;

## 基本使用:
* 水平滑动: 快进快退; <br/>
* 滑动屏幕左侧: 调整亮度; <br/>
* 滑动屏幕右侧: 调整声音; <br/>
* 单机屏幕: 显示或者隐藏控制栏; <br/>
* 双击屏幕: 暂停/播放视频; <br/>
* 点击全屏/退出全屏按钮, 或者旋转屏幕: 进行全屏和普通状态切换;<br/>
* 点击画中画: 进行画中画;
